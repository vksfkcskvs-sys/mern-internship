# Library Management System

## About
This is a simple SQL project to manage a library system.  
It keeps track of books, members, and who borrowed what.

## What it does
- Stores book details  
- Stores member details  
- Tracks borrowing and returning of books  

## Tables used
- books  
- members  
- borrow_records  

## Files
- library.sql → contains all SQL code  
- er-diagram.png → shows database structure  
- screenshots → query outputs  

## Notes
- Database is normalized (no unnecessary duplicate data)  
- Foreign keys are used to connect tables properly
